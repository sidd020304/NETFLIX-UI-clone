# NETFLIX-UI-clone
This Netflix UI Clone is a front-end project built using HTML and CSS that recreates the look and feel of the Netflix homepage.The design uses modern CSS techniques such as Flexbox and Grid for layout management.It is fully responsive and adapts to different screen sizes. No backend functionality or video streaming features are included.
